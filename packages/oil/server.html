サーバー

Oilサーバーを実行すれば、フルスペックのWebサーバーでさえインストールする必要がありません！PHP5.4を組み込んだ Webサーバーを使います。コマンドラインからサーバーを立ち上げ、そしてホストネームとポート番号（デフォルトでは http://localhost:8000）でブラウザから読み込んでください。 

$ php oil server
Listening on http://localhost:8000
Document root is public/
Press Ctrl-C to quit.

使い終わったら、Ctrl-Cでサーバーを終了します。

Fuel をダウンロードする

Fuel は現在バージョン 1.1 です。次のリンクを使用して Fuel をダウンロードできます:

  • リリース
    • Download 1.1
    • Download 1.0.1
    • Download 1.0
  • リポジトリ
    • Fuel on GitHub
    • Fuel Core on GitHub
    • Docs on GitHub
    • Fuel Orm on GitHub
    • Fuel Auth on GitHub
    • Fuel Parser on GitHub
    • Fuel Oil on GitHub
    • Fuel Email on GitHub
  • ユーザ作成の Fuel パッケージ
    • Fuel-packages account on Github
      Send a message to this account or to @fuelphp on Twitter to have your package forked into fuel-packages.

コマンドラインから Fuel の改良されたパッケージの１つをインストールするために Oil ユーティリティ を使用します。

コアに貢献するには

Fuel はコミュニティ主導で開発されており、我々は参加を歓迎しています。もし貢献に興味があれば、 我々のコーディング標準 を読む時間を少し割いてください。 これらのガイドラインを読んでいない貢献者は受け入れられません。読んでいれば、fork, commit, make pull-request することができます。 方法がわからない場合は: http://help.github.com/forking/ を参照してください。

パッケージに貢献するには

他の Fuel ユーザに興味を持ってもらえるものが何かあるでしょうか？ それは パッケージ を作成することです。 適切にコーディングされていれば、コーディング標準 でコンパイルされます。 そして我々はこのページにそれを掲載し、oil コマンドラインユーティリティを使用してのインストールを有効にすることができるでしょう。
